## 📁 Projects

### 🔴 [Predicting Credit Default Risk](https://github.com/arya100-max/My-Portfolio/tree/main/Credit%20Default%20Risk)
> **Jun 2026 | New Delhi Institute Of Management**

Developed an ML model to predict whether a borrower will experience **serious delinquency (90+ days past due)** within two years using historical credit data.

- 📊 Performed EDA, missing value treatment & VIF-based multicollinearity removal
- 🤖 Trained & compared 4 models: Logistic Regression, Decision Tree, Random Forest, Gradient Boosting
- 🏆 **Gradient Boosting achieved 93.63% accuracy** with highest default detection rate
- 🔧 **Tools:** Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
